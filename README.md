# Student Photo Tools

A premium black-and-white React web app for resizing, cropping, compressing, converting, and preparing student photos for government forms.

## Features

- Image resize with aspect-ratio control
- Compress image to a target KB
- Interactive crop with zoom
- Passport and exam photo presets
- Signature resize
- JPG to PNG and PNG to JPG conversion
- Light and dark mode
- Mobile-first responsive UI

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, usually:

```text
http://localhost:5173/
```

## Build

```bash
npm run build
```
